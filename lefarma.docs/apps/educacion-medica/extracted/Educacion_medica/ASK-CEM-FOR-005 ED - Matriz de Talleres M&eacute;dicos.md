## Guía de llenado B

| No. | Concepto | Actividad |
|-----|----------|-----------|
| 1 | Gerencia | Indicar que tipo de gerencia es Centralizado o Descentralizado |
| 2 | Región | Escribir la región donde se encuentra el hospital: sur, norte, poniente, oriente, etc. |
| 3 | Hospital | Escribir el nombre del hospital |
| 4 | Estado | Escribir el estado donde se encuentra ubicado el hospital |
| 5 | Ciudad / municipio | Escribir la ciudad o municipio donde se encuentra ubicado el hospital |
| 6 | No. de participantes | Escribir la cantidad de participantes que asistirán al Taller Médico |
| 7 | Ejecutivo | Escribir el nombre del Ejecutivo de Ventas |
| 8 | Fecha de taller | Escribir la fecha en la cual está programado el taller médico con el siguiente formato: dd/mm/aaaa |
| 9 | Hora | Escribir la hora en la cual se va a realizar el taller médico |
| 10 | ¿Se requiere llevar equipo de proyección? | Escribir "SI" o "NO" según sea el caso |
| 11 | ¿Propio o rentado? | En caso de aplicar, escribir si el equipo de proyección es propio o rentado |
| 12 | Producto | Escribir el nombre del producto |
| 13 | Cantidad (piezas) | Escribir el número de piezas del producto que se van a llevar al taller médico |
| 14 | Cantidad de folletos (piezas) | Escribir el número de piezas de folletos que se van a llevar al taller médico |
| 15 | Costo unitario | Escribir el costo unitario del folleto con el siguiente formato $MXN |
| 16 | Tipo interno o externo | Indicar si el envío lo van a realizar de modo "interno" o "externo" |
| 17 | Costo unitario | En caso de aplicar, escribir el costo unitario del envío con el siguiente formato $MXN |
| 18 | Cantidad (no. de servicios) | Escribir el número de Box Lunch que se van a llevar al Taller Médico |
| 19 | Costo unitario | Escribir el costo unitario del box lunch con el siguiente formato $MXN |
| 20 | Producto | Escribir el costo unitario del producto con el siguiente formato $MXN |
| 21 | Folleto impreso | Es el resultado de la multiplicación del número de piezas de folletos por su precio unitario |
| 22 | Gastos de envío | En caso de aplicar, escribir el costo unitario del envío con el siguiente formato $MXN |
| 23 | Box lunch | Es el resultado de la multiplicación del número de box lunch por su precio unitario |
| 24 | Costo Total | Es el resultado de la suma de: producto + Folleto impreso + gastos de envío + box lunch |
| 25 | Elaboró | Escribir el puesto, nombre y la firma del colaborador que realizó la Matriz de Talleres Médicos. Formato de fecha: dd/mmm/aaaa |
| 26 | Revisó | Escribir el puesto, nombre y la firma del colaborador que revisó la Matriz de Talleres Médicos. Formato de fecha: dd/mmm/aaaa |
| 27 | Autorizó | Escribir el puesto, nombre y la firma del colaborador que autorizó la Matriz de Talleres Médicos. Formato de fecha: dd/mmm/aaaa |
