# ASK-CEM-FOR-002 Base de Datos de Hospitales

> Fuente: `pdf/Educacion_medica/ASK-CEM-FOR-002 ED - Base de datos de hospitales para Talleres Médicos.pdf`
> Código: [[ASK-CEM-FOR-002 Base de Datos de Hospitales|ASK-CEM-FOR-002]] · Versión: 01
> Proceso: Educación Médica · Área: Gerencia de Ventas

Formato para registrar la base de datos de hospitales del sistema público de salud que serán considerados para los [[Talleres Médicos en Hospitales]].

## Encabezado del formato

| Campo | Descripción |
|-------|-------------|
| Gerencia de Ventas | Indicar el tipo de gerencia: IMSS o Descentralizado |
| Con SIA | Marcar si se trata de hospitales con SIA |
| Sin SIA | Marcar si se trata de hospitales sin SIA |
| Año | Escribir el año al que corresponde la base de datos |

## Guía de llenado B

| No. | Concepto | Actividad |
|-----|----------|-----------|
| 1 | Gerencia de ventas | Indicar que tipo de gerencia es IMSS o Descentralizado |
| 2 | Con SIA / Sin SIA | Indicar si se trata de hospitales con SIA o sin SIA |
| 3 | Fecha | Escribir la fecha del año al cual corresponde la base de datos |
| 4 | Subtotales | Escribir los subtotales de las columnas del número de hospitales y de las diferentes tipos de anestesias |
| 5 | Clues | Escribir la clues que le corresponde al hospital |
| 6 | Estado | Escribir el estado donde se encuentra ubicado el hospital |
| 7 | Municipio | Escribir el municipio donde se encuentra ubicado el hospital |
| 8 | Clave de la institución | Escribir la clave del hospital si se trata de IMSS, ISSSTE, SSA Federal, etc. |
| 9 | Nombre del hospital | Escribir el nombre del hospital |
| 10 | Número de quirófanos | Escribir el número de quirófanos con los que cuenta el hospital |
| 11 | Anestesias totales | Realizar el cálculo del número de anestesias totales: **AT = NQ × 2.5 × 250** · NQ = no. de quirófanos del hospital · 2.5 = no. de cirugías promedio por día · 250 = días laborables al año |
| 12 | Anestesias Generales | **AG = AT × 30%** · AT = Anestesias Totales |
| 13 | Anestesias Regionales | **AR = AT × 70%** · AT = Anestesias Totales |
| 14 | Anestesias Epidurales | **AE = AR × 35%** · AR = Anestesias Regionales |
| 15 | Anestesias Subdurales | **AS = AR × 45%** · AR = Anestesias Regionales |
| 16 | Mixtas obesos | **MO = AR × 2%** · AR = Anestesias Regionales |
| 17 | Mixtas no obesos | **MNO = AR × 18%** · AR = Anestesias Regionales |
| 18 | Elaboró | Escribir el puesto, nombre completo y firma del colaborador que realizó la base de datos |
| 19 | Revisó | Escribir el puesto, nombre completo y firma del colaborador que revisó la base de datos |
| 20 | Autorizó | Escribir el puesto, nombre completo y firma del colaborador que autorizó la base de datos |
