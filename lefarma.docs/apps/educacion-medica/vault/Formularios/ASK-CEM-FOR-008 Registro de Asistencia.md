## Guía de llenado B

| No. | Concepto | Actividad |
|-----|----------|-----------|
| 1  | Fecha | Escribir la fecha en la cual se lleva a cabo el taller médico con el siguiente formato: dd-mmm-aaaa |
| 2  | Tema del Taller | Escribir el nombre del producto |
| 3  | Institución | Indique la institución donde se realiza el taller |
| 4  | Unidad Médica | Escribir la hora en que termina el taller Médico |
| 5  | Lugar | Colocar el lugar donde se imparte el taller Médico |
| 6  | Nombre del Especialista | Escribir el nombre completo del especialista de producto |
| 7  | Nombre del Ejecutivo | Escribir el nombre completo del ejecutivo de producto |
| 8  | Especialista de Producto | Escribir el nombre completo del especialista de producto |
| 9  | Nombre | Nombre del Médico |
| 10 | Puesto | Puesto del Médico |
| 11 | Teléfono Celular | Número de contacto del Médico |
| 12 | Correo electrónico | Correo electrónico del Médico |
| 13 | Firma | Firma del Médico que se registró |
| 13 | Observaciones | Registrar el nombre del médico líder, indicando si es un médico líder positivo o negativo del producto |
