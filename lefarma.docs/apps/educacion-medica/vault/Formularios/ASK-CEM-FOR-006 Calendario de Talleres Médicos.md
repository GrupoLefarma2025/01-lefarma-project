# ASK-CEM-FOR-006 Calendario de Talleres Médicos

> Fuente: `pdf/Educacion_medica/ASK-CEM-FOR-006 ED - Calendario de Talleres Médicos.pdf`
> Código: [[ASK-CEM-FOR-006 Calendario de Talleres Médicos|ASK-CEM-FOR-006]] · Versión: 01
> Proceso: Educación Médica · Área: Gerencia de Ventas

Formato mensual para programar y visualizar la agenda de [[Talleres Médicos en Hospitales|Talleres Médicos]] por especialista y ejecutivo.

## Encabezado del formato

| Campo | Descripción |
|-------|-------------|
| Año | Escribir el año correspondiente |
| Mes | Escribir el mes correspondiente |

## Estructura del calendario mensual

El calendario se organiza por semanas. Cada semana ocupa un bloque de filas con la siguiente estructura interna por cada especialista asignado:

| Semana | LUNES | MARTES | MIERCOLES | JUEVES | VIERNES | S | D |
|--------|-------|--------|-----------|--------|---------|---|---|
| **Ciudad** (Especialista 1) | | | | | | | |
| Nombre del especialista 1 | | | | | | | |
| Nombre del ejecutivo de ventas, Hospital y fecha | | | | | | | |
| **Ciudad** (Especialista 2) | | | | | | | |
| Nombre del especialista 2 | | | | | | | |
| Nombre del ejecutivo de ventas, Hospital y fecha | | | | | | | |

> El formato se repite para cada semana del mes. Las columnas S y D corresponden a Sábado y Domingo (días no laborables, sin registro de talleres).

## Guía de llenado B

| No. | Concepto | Actividad |
|-----|----------|-----------|
| 1 | Año | Escribir el año correspondiente |
| 2 | Mes | Escribir el mes correspondiente |
| 3 | Día | Escribir el número que le corresponde a los días de la semana de acuerdo al calendario |
| 4 | Ciudad | Escribir el nombre de la ciudad donde se ubica el hospital |
| 5 | Nombre del especialista | Escribir el nombre del especialista de producto que va a realizar el taller médico |
| 6 | Lugar y hora | Escribir el nombre del ejecutivo de ventas, hospital o lugar que va a visitar y la hora |
